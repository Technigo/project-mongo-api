Documentation for Custom Netflix API

GET endpoints

  • Get all shows

    e.g./api/shows

  • Get a specific show

    e.g./api/shows/60020826

  • Get all shows with a specific actor

    e.g./api/shows?actor=Anna

    e.g./api/shows?actor=Anna Claire Bartlam

  • Get all shows for a specific year

    e.g./api/shows?year=2017

  • Get all shows with a specific duration (min)

    e.g./api/shows?duration=105

  • Get all shows with a specific title

    e.g./api/shows?title=Chocolate

  • Get all shows with a specific director

    e.g./api/shows?director=Stanley Kubrick

  • Get all shows with a specific country

    e.g./api/shows?country=Sweden

  • Get all shows with a specific rating

    e.g./api/shows?rating=TV-14

  • Get all shows with a specific category

    e.g./api/shows?category=Comedies

  • Get all shows with a specific description

    e.g./api/shows?description=In this sequel to the 2016

  • Get all shows with a specific show type

    e.g./api/shows?type=movie

  • Get all movies with pagination

    e.g./api/shows?page=1

  • Get shows with several parameters

    e.g./api/shows?actor=Anna&year=2019

  • Get shows with several parameters

    e.g./api/shows?actor=Anna&year=2019&duration=24

  • Get shows with several parameters (error example)

    e.g./api/shows?actor=Anna&duration=105&page=3
