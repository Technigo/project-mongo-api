Welcome! This is the start page

/shows - view all the data. Filter by title, year, country, actor. (/shows/?year=2019 or /shows/?year=2019&country=south africa) You can also filter by pages and pagesize. Default pagesize is 20. (/shows/?page=1&pageSize=20)

/shows/:id - view all the data for a single movie, filtered by _id (/shows/81193313)

/categories - view all the possible categories

/categories/:category - view all the data in the category. You can also filter the data by pages and pagesize. (/categories/movie/?page=1&pageSize=20)
